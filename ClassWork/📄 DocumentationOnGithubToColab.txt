
📄 Documentation: Reading a Text File from GitHub in Python using Google Colab

🧾 Objective
To read a text file stored in a GitHub repository using Python in Google Colab.

📌 Introduction
GitHub is a platform used to store and manage code repositories. Sometimes, we need to access files (like .txt, .csv, etc.) directly from GitHub into Python for processing.
Google Colab allows us to fetch such files using URLs and Python libraries like requests or urllib.

⚙️ Requirements
* Google Colab environment
* Internet connection
* GitHub repository link containing the text file

🔗 Important Concept: Raw File URL
To read a file from GitHub, we must use the Raw URL of the file.
Steps to get Raw URL:
1. Open file in GitHub repository
2. Click on "Raw" button
3. Copy the URL
Example:
https://raw.githubusercontent.com/user/repo/branch/file.txt

🧑‍💻 Method 1: Using requests Library
📌 Code:
import requests
url = "https://raw.githubusercontent.com/user/repo/branch/file.txt"
response = requests.get(url)
data = response.text
print(data)
✅ Explanation:
* requests.get(url) → fetches file from GitHub
* .text → converts content into readable string

🧑‍💻 Method 2: Using urllib Library
📌 Code:
import urllib.request
url = "https://raw.githubusercontent.com/user/repo/branch/file.txt"
data = urllib.request.urlopen(url).read().decode()
print(data)
✅ Explanation:
* urlopen() → opens file from URL
* .read() → reads content
* .decode() → converts bytes to string

🧑‍💻 Method 3: Cloning the Repository
📌 Step 1: Clone Repo
!git clone https://github.com/user/repo.git
📌 Step 2: Read File
with open("repo/file.txt", "r") as f:
    data = f.read()
    print(data)
✅ Explanation:
* Downloads entire repository
* Reads file like a local file

💡 Example
import requests
url = "https://raw.githubusercontent.com/dwyl/english-words/master/words.txt"
data = requests.get(url).text
print(data[:200])

⚠️ Precautions
* Always use Raw URL, not normal GitHub link
* Ensure file is public (or use authentication for private repo)
* Check internet connectivity

🎯 Conclusion
Reading files from GitHub in Google Colab is simple using Python libraries. The easiest way is using the raw file URL with the requests library, while cloning is useful when working with multiple files.
